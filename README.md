# Customer_Churn_Analysis
# Анализ оттока клиентов телеком-компании

Проект по анализу данных для выявления ключевых факторов оттока клиентов и разработки рекомендаций по увеличению их удержания.

## О проекте

В рамках проекта проведен полный цикл разведочного анализа данных (EDA) на основе синтетического датасета телеком-компании. Целью было определить, какие характеристики клиентов и параметры обслуживания наиболее сильно влияют на решение об уходе, и предложить конкретные меры для снижения оттока.

**Ключевые вопросы исследования:**
1. Каков общий уровень оттока клиентов?
2. Какие факторы наиболее сильно коррелируют с оттоком?
3. Какие сегменты клиентов наиболее уязвимы?
4. Какие бизнес-меры могут снизить отток?

## Стек технологий

- **Python 3** — основной язык программирования
- **Pandas** — обработка и анализ табличных данных
- **NumPy** — численные вычисления
- **Matplotlib/Seaborn** — визуализация данных
- **Jupyter Notebook** — интерактивная среда разработки
